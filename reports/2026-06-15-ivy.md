## 今天做了什么

- 动态详情页支持多图：可滑动 gallery、页码指示、点击进全屏查看器（翻页 + 双指/双击缩放）
- 动态发布性能优化：上传前本地压缩、复用识鱼已上传的图、多图改顺序上传保证全部附加成功
- 帖子删除：作者可在详情页软删自己的帖子，服务端校验归属，删除后从 feed / 我的动态 / 详情各处消失
- 打包流水线：每次发版自动从上一个 tag 起列出增量 commit，按 Feature / Fix 分类写进 GitHub Release notes
- 新增本组织主页 + 工作日报站点（就是你正在看的这个）
